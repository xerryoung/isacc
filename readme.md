# 🥪 ISACC TOAST
이삭토스트 공식 홈페이지를 클론한 프로젝트입니다.<br>
일부분 리컬러 및 수정한 부분이 있습니다.
<br><br>

## 특징
- **반응형 페이지** : 최소너비 768px ~ 1440px 이상
<br><br>

### 파일 구조
```
├ css               css 파일
├ img               이미지 파일
├ js                자바스크립트 파일
│
├ index.html        메인 페이지
├ sb2_menu.html     세부 페이지: 메뉴
├ sb3_notice.html   세부 페이지: 소식
├ sb6_cs.html       세부 페이지: 고객센터
```
<br>

## 기술 스택
- html5 : 시맨틱 마크업
- css3 : position, flex, 애니메이트
